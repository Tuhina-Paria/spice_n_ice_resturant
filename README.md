# 🍽️ Spice 'n' Ice — Restaurant Website

A modern, mobile-first restaurant website built to showcase menu, gallery, and customer experience with a premium UI.

## 🚀 Live Demo
👉 https://your-vercel-link.vercel.app

## ✨ Features

- 📱 Fully responsive (mobile-first design)
- 🎬 Smooth animations & page transitions
- 🍽️ Interactive dishes section
- 📸 Gallery with lightbox preview
- ⭐ Customer reviews section
- 📍 Location & contact page
- 🛒 Order integration (Zomato & Swiggy ready)
- 🎨 Premium UI with gradients & glass effects

---

## 🛠️ Tech Stack

- ⚛️ React (Vite)
- 🎨 Tailwind CSS
- 🎞️ Framer Motion
- 🔗 React Router
- 🎯 Lucide Icons
